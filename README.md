# ❌⭕ Circle Cross (Java)

A customizable console-based implementation of the classic Tic-Tac-Toe game, built in Java using Object-Oriented Programming (OOP).

## 📖 Description

This project is an extended version of the traditional Tic-Tac-Toe game. It allows players to configure game rules before starting and supports dynamic board sizes and win conditions.

The game is divided into three main stages: configuration, gameplay, and results.

## ⚙️ Game Stages

### 1. 🔧 Configuration

Before the game starts, players set up the rules:

* Player A enters their name and chooses a symbol (e.g., X or O)
* Player B enters their name and selects a symbol
* Players define how many symbols in a row are required to win
* Players choose the board size (horizontal and vertical)

### 2. 🎮 Gameplay

* Players take turns selecting positions on the board
* Moves are made by entering a number corresponding to a board position
* The starting player is chosen randomly by the program
* Players alternate turns until the game ends
* Players can exit the game at any time by typing `quit`

### 3. 🏆 Results

* When a player achieves the required number of symbols in a row, they win the round
* The winner receives a point
* The scoreboard is updated and displayed
* A new game starts automatically
* The starting player alternates (the player who went second in the previous round starts next)

## 🚀 Features

* ❌⭕ Customizable Tic-Tac-Toe rules
* 📏 Adjustable board size
* 🎯 Configurable win condition (number of symbols in a row)
* 🎲 Random starting player
* 🔄 Alternating starting player between rounds
* 🏆 Score tracking system
* ⚙️ Object-Oriented design
